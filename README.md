# 3D T-Shirt Designer

A proof-of-concept 3D T-shirt designer app built with **THREE.js**.

Best viewed in desktop.

## Features

- Change T-shirt base color.
- Add / edit text with multiple styling options.
- Add / edit images on the T-shirt surface using filters and transforms.
- Support for an arbitrary number of font sizes.
- Export the final design as a 3D model in **GLFT** format.

## Technologies Used

- **3D Graphics**: [THREE.js](https://threejs.org/)
- **Color Picker**: [@simonwep/pickr](https://github.com/Simonwep/pickr)
- **Build Tool**: [Vite](https://vitejs.dev/)
